# App Assets Needed for Submission

You need to create these image assets for your app. You can hire a designer on Fiverr for $50-100 to create all of these.

## Required Assets

### 1. App Icon (CRITICAL)
**File:** `assets/icon.png`
- **Size:** 1024x1024 pixels
- **Format:** PNG with transparency
- **Design:** Your app logo, simple and recognizable
- **Used for:** Home screen icon on phones

**Quick option:** Use a free tool:
- https://appicon.co/
- https://makeappicon.com/
- https://icon.kitchen/

### 2. Splash Screen
**File:** `assets/splash.png`
- **Size:** 1242x2436 pixels (iPhone resolution)
- **Format:** PNG
- **Design:** Your logo centered on solid background
- **Used for:** Loading screen when app opens

### 3. Adaptive Icon (Android)
**File:** `assets/adaptive-icon.png`
- **Size:** 1024x1024 pixels
- **Format:** PNG with transparency
- **Design:** Same as app icon but with padding
- **Note:** Android crops icons in different shapes

### 4. Notification Icon (Optional but recommended)
**File:** `assets/notification-icon.png`
- **Size:** 96x96 pixels
- **Format:** PNG, white icon on transparent background
- **Design:** Simple, recognizable symbol
- **Used for:** Push notifications

### 5. Favicon (for web version)
**File:** `assets/favicon.png`
- **Size:** 192x192 pixels
- **Format:** PNG
- **Design:** Small version of your logo
- **Used for:** Browser tab icon

## How to Create Screenshots

### Option 1: Use Your Phone (Free)
1. Run the app: `npm run mobile`
2. Open on your phone via Expo Go
3. Navigate to each screen
4. Take screenshots (Power + Volume Down)
5. Upload to computer

### Option 2: Use iOS Simulator (Mac only)
1. Run: `npm run mobile:ios`
2. Wait for simulator to open
3. Navigate to each screen
4. Cmd + S to save screenshot

### Option 3: Use Android Emulator
1. Run: `npm run mobile:android`
2. Navigate to each screen
3. Click camera icon in emulator toolbar

### Option 4: Hire Designer (Recommended)
Cost: $100-200
- They create perfect screenshots
- Add marketing text overlay
- Professional looking
- All sizes included

## Quick Start - Use Placeholders

If you want to test the build process first, you can use placeholder images temporarily:

1. Create simple colored squares in any image editor
2. Save at the required sizes
3. Test the build process
4. Replace with real assets before public launch

**Don't submit to stores with placeholders!** Apple and Google will reject your app.
